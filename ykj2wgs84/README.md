This Python script can be used to convert YKJ geographical coordinates to WGS84 degree coordinates.

You need to change the code to get the script to work for your data.

## Author

Teemo Tebest (Yle), @teelmo @IRCnet

## USAGE

python ykj2wgs84.py

## Required libraries

- pymongo
- minidom
- csv
- urllib
