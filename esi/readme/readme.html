HTML

Wräppää upotettava merkkaus yhden elementin sisään. Anna elementille id, joka on muotoa esi-toteutuksen-nimi. Esimerkiksi näin:

<div id="esi-testi">
</div>

Javascript

Käytämme tällä hetkellä jQueryn 1.9-versiota. Älä lataa upotuksen mukana uudempaa tai vanhempaa versiota, sillä se voi rikkoa palvelun omat komponentit. Voit käyttää toteutuksessa harkiten Cookie- ja hashchange-plugaria. Käytä jQueryä seuraavan esimerkin mukaisesti:

window.esiTesti = window.esiTesti || {};
window.esiTesti.init = function() {
  $(function() {
    var $esiTesti = $('#esi-testi');
    $esiTesti.append('<p>jQuery '+$(this).jquery+'</p>')
  });
};
esiTesti.init(jQuery);

CSS

Voit käyttää skaalautuvuuden toteuttamiseen media queryn min-width -ominaisuutta arvoilla 480px, 767px, 980px ja 1260px. Aloita jokainen selektori toteutuksesi id:llä, esimerkiksi näin:

#esi-testi {
  clear: both;
  margin-bottom: 20px;
  position: relative;
}
#esi-testi > p {
  background: #000;
  color: #fff;
  padding: 10px;
}
@media screen and (min-width: 980px) {
  #esi-testi > p {
    background: #333;
  } 
}

Vaatimukset ennen julkaisua

  • Toteutus skaalautuu mobiilileveyksistä desktopleveyksiin.
  • Toteutus toimii vähintään IE:n 7-9 -versioilla sekä uusimmalla Firefox-, Chrome- ja Safari-selaimella.
  • Toteutus toimii myös kosketusnäytöllä, esimerkiksi iPadillä.
